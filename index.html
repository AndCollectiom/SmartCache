A caching extension for Retrofit 2!

Introduction

SmartCache improves caching support for Retrofit 2:

  • Memory-based caching
  • Instant cache hit (response before the call is even started)
  • Adapter/extensibility support

To add it to your project, add the SmartCallFactory: 
SmartCallFactory smartFactory = new SmartCallFactory(BasicCaching.fromCtx(this));
Retrofit retrofit = new Retrofit.Builder()
    .baseUrl("http://myapi.com")
    .addCallAdapterFactory(smartFactory)
    .build();
You should also switch from Call<T> to SmartCall<T> on the calls you want to cache: 
public interface GitHubService {
  @GET("/users/{user}/repos")
  SmartCall<List<Repo>> listRepos(@Path("user") String user);
}

Limitations

At the moment, this library doesn't support:

  • Rx calls
  • execute() (blocking calls) which don't make sense
